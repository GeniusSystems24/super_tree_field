# Requirements and Issues

## Editable Mode

- يمكن تغيير اسم العقد
- يمكن اضافة عقدة اخ قبل وبعد
- يمكن اضافة عقدة ابن
- يمكن تغيير موقع العقدة وابناءها من خلال السحب والافلات
- يمكن حذف عقدة وابناءها

## Readable mode

### Node context menu

- يمكن اضافة قائمة تظهر عند الضغط على العقدة باليمين
- فتح العقدة واقفالها من خلال الضغط على العقدة
